## ‚ö†Ô∏è IMPORTANT: You need Node.js installed first!

### Step 1: Install Node.js (5 minutes)

1. **Open your web browser**
2. **Go to:** https://nodejs.org/
3. **Click the big green "Download Node.js (LTS)" button**
4. **Open the downloaded file** (it will be in your Downloads folder)
5. **Follow the installation wizard** (just click "Next" or "Continue" on everything)
6. **When it says "Installation complete" - you're done!**

### Step 2: Restart Your Terminal

- **Close your Terminal app completely**
- **Open it again** (Cmd+Space, type "Terminal")

### Step 3: Start the Server

Copy and paste these commands ONE AT A TIME into Terminal:

```bash
cd ~/Desktop/Cursor/Portfolio
```

Press Enter, then:

```bash
npm install
```

Press Enter and wait for it to finish (this might take a minute), then:

```bash
npm start
```

### Step 4: You Should See:

```
Server running on http://localhost:3000
```

**‚úÖ SUCCESS!** The server is now running!

### Step 5: Use the Admin Panel

1. **Open your web browser**
2. **Go to:** http://localhost:3000/admin.html
3. **You should see a green "‚úì Server connected" message**
4. **Enter password:** `admin123`
5. **Click "Access"**
