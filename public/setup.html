GitHub Setup Wizard

Follow these steps to connect your portfolio to GitHub

1

Use Your Existing GitHub Repository

You can use an existing repository (like portfolio) or create a new one:

  • Using existing repo: Just make sure you know your GitHub username and the repo name
  • Creating new repo: Go to github.com/new and create one

Note: The portfolio will create an images/ folder and a content.json file in your repo. If these already exist, they'll be updated.

2

Create a Personal Access Token

You need a token to upload files to GitHub:

  1. Go to GitHub Token Settings
  2. Give it a name like "Portfolio Admin"
  3. Select Expiration (90 days, 1 year, or no expiration)
  4. Check the repo scope (this gives full repository access)
  5. Scroll down and click "Generate token"
  6. IMPORTANT: Copy the token immediately (it starts with ghp_) - you won't see it again!
